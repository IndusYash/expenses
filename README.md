# Personal Finance Tracker

A modern, student-friendly personal finance tracking application built with React and Tailwind CSS.

## Project Structure

```
expenses/
├── frontend/          # Frontend React application
│   ├── src/           # Source code
│   ├── package.json   # Dependencies
│   └── README.md      # Frontend documentation
└── README.md          # This file
```

## Getting Started

Navigate to the frontend directory and follow the instructions in `frontend/README.md`:

```bash
cd frontend
npm install
npm run dev
```

For detailed documentation, see [frontend/README.md](./frontend/README.md)
